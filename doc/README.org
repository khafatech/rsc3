#+title: rsc3 documentation

The documentation for rcs3 can be found in the =rcs3-docs= package or online at [[https://github.com/zzkt/rsc3-docs][github]]

Documentation includes examples & help files whichcorrespond closely to the SuperCollider help files included in the standard distribution. Examples from the help files, tutorials and other sources are also included.  Therre is also an rsc3 specific  tutorial inluded,
[[https://rohandrape.net/ut/rsc3-texts/html/rsc3-tutorial.html][A brief introduction to using scheme supercollider]], by Rohan Drape.
